# G-Est

This is a Repo that puts G-Est_frontend and G-Est_backend together

## Usage

``` shell
# Clone repos using "dev" branch and installs dependencies
./commands.sh run setup
# Start
./commands.sh start
# Migrate backend
./commands.sh run migrate
# Stop
./commands.sh stop
```

## Start

``` shell
./commands.sh run setup && \
./commands.sh start && \
./commands.sh run migrate
```

## Stop

``` shell
./commands.sh stop
```
